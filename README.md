# plug_server

API backend em TypeScript com Express e Socket.IO.

## Stack

- Node.js + TypeScript
- Express (REST)
- Socket.IO (tempo real)
- Zod (validacao)
- JWT access + refresh token rotation
- Vitest + Supertest

- [CHANGELOG.md](./CHANGELOG.md) — mudancas, migracao e roadmap tecnico
- [docs/api_rest_bridge.md](./docs/api_rest_bridge.md) — contrato HTTP/Socket do bridge de comandos

## Scripts

- `npm run lint` - lint
- `npm run test` - testes
- `npm run build` - build de producao
