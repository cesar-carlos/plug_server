# plug_server

API backend em TypeScript com Express e Socket.IO.

## Stack

- Node.js + TypeScript
- Express (REST)
- Socket.IO (tempo real)
- Zod (validacao)
- JWT access + refresh token rotation
- Vitest + Supertest

## Scripts

- `npm run dev` - desenvolvimento
- `npm run typecheck` - checagem de tipos
- `npm run lint` - lint
- `npm run test` - testes
- `npm run build` - build de producao
